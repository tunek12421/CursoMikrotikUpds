Preguntas Frecuentes sobre MTCNA

¿Qué es el curso de certificación MikroTik MTCNA?
Es un programa de formación que prepara a los estudiantes para obtener la certificación MTCNA de MikroTik. Esta certificación demuestra que el titular tiene un conocimiento profundo de las redes y dispositivos MikroTik y está capacitado para configurar, administrar y mantener redes MikroTik.
¿Cuánto tiempo dura el curso de Certificación MTCNA?
Este curso suele tener una duración de 24 horas repartidos en varios días, aunque esto puede variar según la modalidad de enseñanza (presencial o online).
¿Qué conocimientos básicos se requiere para tomar un curso de certificación MTCNA?
Debes tener buenos conocimientos de subnetting, VLSM, modelo OSI, modelo TCP/IP, conceptos básicos de redes LAN, redes wireless. También necesitas conocer qué es una dirección IP, máscara de red, Gateway y DNS.
¿Cómo puedo obtener la certificación MikroTik MTCNA?
Para obtener la certificación MTCNA, debes completar el curso de capacitación y aprobar el examen de certificación con una nota mínima del 60%.
¿Cuál es la nota mínima para aprobar el Examen de Certificación de MTCNA?
La nota mínima es 60% sobre 100%. Si se obtiene entre 50% y 59% el estudiante tiene derecho a una segunda oportunidad para rendir un nuevo examen.
¿Se puede tomar el curso de certificación de MTCNA de forma ON-LINE?
Sí, desde 2020 MikroTik autorizó a todos los Trainers poder dar los Cursos de Certificación de manera ON-LINE. Sin embargo, el examen de certificación debe rendirse de manera presencial.
¿Puedo rendir remotamente el examen de certificación MTCNA?
Para primera vez, el examen debe rendirse presencial frente al Trainer. Para renovaciones, se puede rendir de manera remota a través del sistema MTCOPS implementado por MikroTik.
